# Anime Jupiter

Anime Jupiter is a program for view your personal list of Anime

# Features:
- Admin / User Side
- Admin Anime CRUD operations
- Search bar
- Scroll bar
- Sort options
- Image chooser
- Scroll label for alert
- Open anime trailer
- Crunchyroll link
